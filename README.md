# huntfiles
a tool to search files

# execute :  
java -jar dist/huntfiles-0.1-boot.jar

java -jar dist/huntfiles-0.1-boot.jar --dir "c:/temp" --text "hello you" --file "*.txt" --zip

--help for options.


# or 
(context menu) huntfiles-0.1-boot.jar ,  open with java
